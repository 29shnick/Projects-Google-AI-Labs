# Google AI Professional Certificate: Applied Labs & Capstone

## Overview
A collection of over 20 hands-on AI projects and labs completed as part of the Google AI Professional Certificate. This repository demonstrates practical experience in applied AI workflows, data analysis, research synthesis, and creative content generation.

## Key Competencies Demonstrated
* **Prompt Engineering:** Designing robust prompts for complex reasoning, data extraction, and creative tasks.
* **AI Workflows:** Automating and scaling tasks across research and data analysis using AI tools.
* **"Vibe Coding" & Natural Language Programming:** Using AI-assisted development to rapidly prototype applications.

## Capstone Project Highlights
The culminating project for this certification involved building a custom AI-powered application. 
* **Tools Used:** Google AI Studio, Cloud Run.
* **Core Skills:** API integration, model selection, and end-to-end deployment of an AI-native app.
